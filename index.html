CSS.

CSS permite crear reglas que especifican cómo debe aparecer el contenido de un elemento. Por ejemplo, puede especificar que el fondo de la página sea crema, que todos los párrafos aparezcan en gris con el tipo de letra Arial o que todos los encabezados h1 estén en un tipo de letra etc.

  1. Reglas de CSS.

    • SELECTOR:

      Indica a qué elemento se aplica la regla. La misma regla puede aplicarse a más de un elemento si separa los nombres de los elementos con comas.

    • DECLARACION:

      Indica cómo los elementos se debe aplicar a un estilo al que se hace referencia en el selector. Las declaraciones se dividen en dos partes (una propiedad y un valor)

    • PROPIEDAD:

      Indica los aspectos del elemento que desea cambiar. Por ejemplo, color, fuente, ancho, alto y borde.

    • VALOR:

      Especifica la configuración que desea utilizar para las propiedades elegidas.

  2. Conceptos de CSS - Ejemplos

    • SELECTORES :
      1. Selector de clase: .claseMaestro
      2. Selector de ID: #idAlumno
      3. Selector Universal: *
    • ESPECICIFIDAD :
      1. etiqueta: 1
      2. clase: 10
      3. id: 100
      4. inline: 1000
